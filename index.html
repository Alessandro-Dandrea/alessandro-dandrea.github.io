Why this thing now?

This is just a very raw website created directly using GitHub pages.
I don't need a super fancy dynamic website since I already have a Substack in Italian, one in English, and my super LinkedIn account.
So this is just a project I've set up on a Saturday afternoon instead of watching anime. Yes, I enjoy having boring Saturday afternoons...

Furthermore, I wanted to test something like GitHub Pages or Cloudflare Pages for years.

And well, in fact this is exactly intended primary as a testing website for Cloudflare pages. So please expect to find A LOT of broken things around.

Eventually, I opted for a pure coding experience with some help from VSCode and Cursor.

My desire to be more confident and refresh my HTML and CSS was stronger.
