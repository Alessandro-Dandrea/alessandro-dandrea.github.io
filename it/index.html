Perché questa cosa adesso?

Questo è solo un sito web molto grezzo creato direttamente usando GitHub pages.
Non ho bisogno di un sito web dinamico super elaborato dato che ho già un Substack in italiano, uno in inglese e il mio super account LinkedIn.
Quindi questo è solo un progetto che ho messo in piedi in un sabato pomeriggio invece di guardare gli anime. Sì, mi piace passare dei noiosi sabati pomeriggio...

Inoltre, volevo testare qualcosa come GitHub Pages o Cloudflare Pages da anni.

E in effetti questo è inteso principalmente come un sito web di test per Cloudflare pages. Quindi aspettatevi di trovare UN SACCO di cose rotte in giro.

Alla fine, ho optato per un'esperienza di pura programmazione con un po' di aiuto da VSCode e Cursor.

Il mio desiderio di essere più sicuro di me e rinfrescare il mio HTML e CSS è stato più forte.
